  • tuidFactoryCtor(randomFillSync: (buffer: Buffer, offset: number, count: number) => void, nowMs: () => number, formatter?: undefined | tuidFormatterType): () => string
    • Defined in lib/tuid.ts:44

    Returns a 128bit time prefixed (48 bits) random (80 bits) identifier encoded as hex (by default) Suitable for distributed system identifier generation. Hard to guess (sparse), increasing security Semi-monotonic, avoiding index performance issues seen by purely random identifiers Ideally stored as a fixed length binary field in the DB. If using a varchar or transmitting via a string the more compact base64url encoding should be used.
